# 物料开发接入流程

## 需求评审

### 前置环节评审（产品、UED）

- 输出需求文档
- 输出 UE 设计稿

## 业务分析及物料定性

### 业务分析

- 确定业务场景
- 确定关键 API 设计

### 技术分析

- 确定关键依赖分析
- 技术评审文档

### 物料定性

- 确定物料类型，所属空间
- 确定物料使用方式

## 授权开发准入

- 开发仓库代码授权
- NPM 发布空间授权
