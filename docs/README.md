### Install

全局安装`@winfe/winex-cli`脚手架工具，用来安装物料。111

```bash
$ yarn global add @winfe/winex-cli
```

### Usage

通过脚手架指令安装业务组件(component)和初始化模板(scaffold)。

```bash
$ winex add --plugin @winex-comp/demo
$ winex init --name my-scaffold
```
